## Workflow

**1. Initialize terraform working directory**

```shell script
terraform init
```

**2. Validate configuration**

```shell script
terraform validate
```

**3. Generate execution plan**

```shell script
terraform plan
```

**4. Format configuration**

```shell script
terraform fmt
```

**5. Apply configuration**

```shell script
terraform apply
```

**5. Destroy resources**

```shell script
terraform destroy
```

**Generate plan and save it to file**

```shell script
terraform plan -out <file>
```
